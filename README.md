# 📊 HR Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing HR data to understand employee attrition, workforce distribution, and key factors affecting employee retention.

The dashboard provides actionable insights to help organizations improve decision-making and reduce attrition.

---

## 🎯 Key Metrics
- Total Employees: 1,417
- Active Employees: 1,186
- Attrition Count: 231
- Attrition Rate: 16.30%
- Average Age: 36.94

---

## 🔍 Key Insights
- High attrition observed in certain departments
- Salary plays a major role in employee turnover
- Job satisfaction strongly impacts retention
- Mid-level experience employees show higher attrition trends

---

## 📊 Dashboard Features
- Attrition by Department
- Attrition by Salary Slab
- Job Role vs Job Satisfaction
- Age Group Distribution
- Gender Analysis
- Experience-based Attrition Trends

---

## 🛠 Tools & Technologies
- SQL – Data Extraction & Cleaning
- Power BI – Data Visualization
- DAX – KPI Calculations
